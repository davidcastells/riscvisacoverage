Behaviour: C.ADDI4SPN is a CIW-format RV32C/RV64C-only instruction that adds a zero-extended non-zero immediate, scaled by 4, to the stack pointer, x2, and writes the result to rd0. This instruction is used to generate pointers to stack-allocated variables, and expands to addi rd0, x2, nzuimm[9:2].
